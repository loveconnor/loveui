## Overview

Use Toggle Group when several toggle buttons share one state model.

Toggle groups are useful for text alignment, formatting controls, view modes, filters, and segmented choices.

## Anatomy

A toggle group contains toggle items. It can allow one selected item or multiple selected items.

## Behavior

Use single selection when the choices are mutually exclusive. Use multiple selection when each toggle is independent but visually grouped.

## Accessibility

Each toggle needs an accessible name. Icon-only toggle groups should use labels, tooltips, or accessible text so users know what each item does.
